This repository contains the nodejs server that runs the naive baiyes classifier in the local machine
this classifier is called in chrome extension as api end point at http://localhost:3000/classify --> refer the server.js for complete details


COMMANDS :

cmd for libraries and dependency , server start

1. npm init -y
2. npm i natural express body-parser
3. npm i papaparse
4. npm install cors
5. node server.js
